observations

  • From an eight chain, "circulate once and a half" is the same as "circulate a half, circulate a half".
  • From facing lines, 1/2 stable regroup = hocus pocus.
  • It's possible to define zip code 0 so that it fits the pattern: centers 1/4 out and 1/2 run (i.e. the ends do their part of slim down, and the centers either phantom run or slim down).
  • The word "spin" has many meanings:
    • swing, e.g. @spin the top@ vs @C3A|the top@
    • swing slip, e.g. @spin the windmill@ vs @the windmill@
    • swing generous slip, e.g. @spin chain thru@ vs @C1|chain thru@
    • cast off 3/4, e.g. @rotary spin@ vs @rotary@
    • touch 3/4, e.g. @spin the pulley@ vs @the pulley@
